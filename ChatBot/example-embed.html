NeighbourBot Embed Example

How to Embed

You can embed NeighbourBot in your application using an iframe:

<iframe src="http://localhost:3001/?token=YOUR_JWT_TOKEN&neighbourhoodId=NEIGHBOURHOOD_ID&apiBase=/api/v1" width="100%" height="600px" frameborder="0" ></iframe>

Configuration Options

  • token: JWT authentication token (required)
  • neighbourhoodId: Current user's neighbourhood ID
  • apiBase: API base URL (default: /api/v1)

JavaScript Integration

You can also configure the chatbot via JavaScript:

// Set configuration before loading iframe window.neighbourbotToken = 'your-jwt-token'; window.neighbourbotApiBase = '/api/v1'; window.neighbourbotNeighbourhoodId = 'neighbourhood-uuid'; // Listen for neighbourhood updates window.addEventListener('message', (event) => { if (event.data.type === 'neighbourhood_updated') { console.log('Neighbourhood updated:', event.data.neighbourhoodId); } });
